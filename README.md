# Motion Practice

## 🚀 Overview

A curated collection of high-performance, reusable UI components built with a focus on fluid motion and interactive design. This project serves as a playground for practicing complex animation sequences, gesture-based interactions, and layout transitions.

## 🛠 Tech Stack

Framework: React.js

Animation Library: Motion

Styling: Tailwind CSS

Language: Javascript

## 📦 Installation

To get this project running locally, follow these steps:

#### 1. Clone the repository:

```bash
git clone https://github.com/mahim-dewan/motion-animations.git
```

#### 2. Install dependencies:

```bash
npm install
# or
yarn install
```

#### 3. Run the development server:

```bash
npm run dev
```
